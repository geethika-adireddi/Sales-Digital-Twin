# Sales Acceleration using Digital Twin

## Project Overview
This project simulates a sales system using a digital twin approach to improve revenue.

## Features
- Simulates daily leads
- Compares normal vs optimized sales
- Visualizes revenue growth

## Files
- main.py → simulation code
- Sales_Acceleration_Digital_Twin_report.docx → project report
- outputs → result graph
